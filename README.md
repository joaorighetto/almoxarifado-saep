# SAEP Almoxarifado

Sistema Django para controle de materiais, saídas e sincronização de planilha de controle.

## Começando

```bash
source .venv/bin/activate
make bootstrap
make run
```

## Comandos úteis

```bash
make help
make dev
make qa
make lint
make format
make test
make test-material-search
```
